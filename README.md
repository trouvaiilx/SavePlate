# SavePlate

SavePlate is a comprehensive web application designed to help users manage their food inventory, reduce food waste, and facilitate food donations within their community. Developed for the **BIT216 - Software Engineering Principles** course.

## Project Structure

This repository is organized into three main parts:

- `/client`: The frontend application built with React, Vite, and Tailwind CSS.
- `/server`: The REST API backend built with Node.js and Express.
- `/database`: The MySQL database schema and related setup scripts.

## Tech Stack

- **Frontend**: React (TypeScript), Vite, Tailwind CSS, Radix UI Components
- **Backend**: Node.js, Express.js
- **Database**: MySQL 8.x

## Features

- **User Authentication**: Secure login with 2FA support and email verification.
- **Inventory Management**: Track food items, quantities, and expiry dates.
- **Donation System**: Share available food with the community or claim donations from others.
- **Meal Planning**: Plan meals for the week using available ingredients.
- **Notifications**: Alerts for expiring food, donation updates, and meal reminders.

## Getting Started

Please refer to the individual `README.md` files in the `/client`, `/server`, and `/database` directories for specific setup and execution instructions.
